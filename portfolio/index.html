01

AgniSena — IoT Forest Fire Detection

A hybrid CNN-FFNN architecture deployed on IoT sensor networks for real-time predictive hazard analysis. Patent filed; awarded the grand prize at the India AI Impact Summit 2026.

CNNFFNNPythonIoTTensorFlow

Problem

Forest fires escalate quickly, and conventional detection systems rely on visual confirmation after ignition. We needed a way to predict and alert before the fire spreads.

Approach

  • Hybrid CNN-FFNN fusing visual + sensor streams
  • Real-time inference on low-power edge devices
  • Integrated alert pipeline to field responders

Impact

  • Ranked 1st of 20,000+ projects, YuvAI 2026
  • ₹15,00,000 grand prize (Team AgniSena)
  • Patent application filed for the hybrid architecture
  • Covered by PIB, NDTV, The Statesman, and 20+ outlets

Stack

Python, TensorFlow, Keras, IoT sensor suite, edge deployment.

02

Driver Drowsiness Detection System

Real-time drowsiness monitor using OpenCV facial landmarks on Raspberry Pi and Arduino Uno, with a Streamlit dashboard and buzzer alerting hardware for at-risk drivers.

OpenCVdlibRaspberry PiArduinoStreamlit

Problem

Drowsy driving causes thousands of preventable accidents each year. Commercial solutions are expensive and proprietary — I built an affordable open alternative.

Approach

  • 68-point facial landmarks via dlib for eye-aspect-ratio tracking
  • Raspberry Pi inference + Arduino Uno for hardware I/O
  • Streamlit dashboard for live telemetry and logs
  • Buzzer + haptic alerts when EAR drops below threshold

Stack

Python, OpenCV, dlib, Streamlit, Raspberry Pi 4, Arduino Uno, piezo buzzer.

Outcome

Reliable real-time detection on an edge device with a total hardware cost under ₹5,000.

03

TruthTriage — Medical RAG System

A retrieval-augmented generation framework that grounds LLMs in verified medical datasets to reduce hallucinations. Awarded 2nd Runner Up at IMI Ygnite 2026.

LLaMA 2LangChainFAISSFastAPIRAG

Problem

General-purpose LLMs hallucinate on clinical questions — dangerous for any triage-adjacent use. TruthTriage grounds every response in verified medical literature.

Approach

  • Curated medical corpus with FAISS vector indexing
  • LangChain orchestration with source citation
  • LLaMA 2 backbone, served via FastAPI
  • Confidence scoring and abstention on low-support queries

Recognition

  • 2nd Runner Up, IMI Ygnite InnovateX, IgniteX 2.0
  • Team Hackspire — IMI Kolkata, Jan 2026
  • Cash prize ₹5,000

Stack

LLaMA 2, LangChain, FAISS, FastAPI, Python.
